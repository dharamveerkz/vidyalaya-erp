# 🎓 Vidyalaya ERP

A modern, minimalistic School Management System built with Next.js 16, Tailwind CSS, and Prisma.

## ✨ Features

- **Role-Based Access Control**: Admin, Teacher, and Student dashboards.
- **Modern Stack**: Next.js 16 App Router, React 19, TypeScript.
- **Authentication**: Secure login with NextAuth v5 & Bcrypt.
- **Database**: Prisma ORM with type-safe queries (SQLite dev / PostgreSQL prod).
- **UI**: Clean, accessible components using Tailwind CSS.
- **Server Actions**: Mutations handled securely on the server side.

## 🚀 Getting Started

### 1. Prerequisites

- Node.js 18+
- npm, yarn, or pnpm

### 2. Installation

```bash
git clone https://github.com/your-username/vidyalaya-erp.git
cd vidyalaya-erp
npm install
```
